## ✨ Features

- 🎨 Modern UI with gradients and clean typography  
- 📱 Fully responsive (mobile-first approach)  
- ⚡ Smooth animations using `@keyframes`  
- 🔄 3D card flip effects using CSS transforms  
- 🎥 Background video integration  
- 🧭 CSS-only navigation menu (checkbox hack)  
- 💬 Popup modal using `:target`  
- 🧩 Scalable architecture using BEM methodology  

---

## 🛠️ Tech Stack

- HTML5  
- SCSS (Sass)  
- CSS3  
- Ionicons  
- Google Fonts (Lato)
